## 1. 静的ファイルのホスティング

<!--
ブラウザでアクセスしたときの「画面」はどこから来ているのでしょうか？
-->

---

ターミナルで一度 `ctrl` + `c` でサーバーを停止してから、
`cat static/index.html` と打って中身を見てみましょう

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>DenoでシンプルなBBS</title>
...
```

ブラウザはこのファイルを受け取って画面を描画しています。これを **「静的ファイルホスティング」** と呼びます。

---

## 2. APIの作成

<!--
では、書き込んだデータや過去の投稿は、どうやってやり取りしているのでしょうか？
-->

---

ターミナルで `cat server.js` と打って
サーバー側のプログラムを見てみましょう

```js
Deno.serve(async (req) => {
  const url = new URL(req.url);

  if (req.method === "GET" && url.pathname === "/api/list") {
    return Response.json(data);
  }
  if (req.method === "POST" && url.pathname === "/api/add") {
    data.push(await req.json());
    return new Response("ok");
  }
```

<!--
ここが「API（Application Programming Interface）」と呼ばれる部分です。
`/api/list` にアクセスされるとデータを返し、`/api/add` にデータが送られてくるとそれを保存する処理が書かれています。
-->
